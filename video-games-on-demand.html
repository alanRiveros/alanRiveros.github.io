Video Games On Demand

Alanriveros | lun. 04 febrero 2019

Capturador de pantalla con python y opencv

Acabo de ver una aplicación que me llamó un poco la atención, se llama
GeForce NOW la aplicación aún está en beta pero propone un concepto muy interesante Video Juegos on demand, algo así como netflix pero de video juegos, lo cuál me pone bastante feliz porque es algo que me parecía raro que nadie hubiera intentado.
Lo que propongo en este post es hacer un clon muy modesto (no me quiero dejar la vida en esto). Por supuesto esto es una aplicación de tipo cliente-servidor y para no volverme loco
con las compatibilidades de los juegos va a correr sobre windows.
Voy a hacer un pequeño listado de las cosas que creo necesarias para este proyecto.

  • Un servidor que corra el juego
  • Capturar la ventana del juego
  • Encodear las imagenes que se capturan y meterlas en un buffer
  • Stremear las imagenes encodeadas por medio de sockets
  • Recibir comandos de teclas presionadas y reproducirlas en la ventana que se está capturando
  • El cliente debe conectarse al servidor por medio de un puerto (cuack)
  • Recibir el stream de datos y mostrarlo en pantalla
  • Enviar comandos de teclas presionadas

Primero voy a empezar con el servidor que captura la ventana del juego, solo porque me pareció
algo divertido, el codigo lo pego a continuación y luego paso a detallarlo.

import numpy as np
import cv2
from PIL import ImageGrab
from winguiauto import findTopWindow
import win32gui
import win32con

hwnd = findTopWindow("<Nombre Programa>")
win32gui.SetWindowPos(hwnd, win32con.HWND_TOPMOST, 0,0,0,0,
win32con.SWP_NOMOVE | win32con.SWP_NOSIZE)


rect = win32gui.GetWindowPlacement(hwnd)[-1]
print(rect[2:4])

while(True):
    img = ImageGrab.grab(bbox=rect) #x, y, w, h
    img_np = np.array(img)
    cv2.imshow("frame", img_np)
    key = cv2.waitKey(1)
    if key == 27:
        break

#vid.release()
cv2.destroyAllWindows()

usamos numpy para el manejo de bitmaps, cv2 para mostrar los frames, ImageGrab para capturar la pantalla, winguiauto es un modulo wrapper de win32gui, lo uso para ubicar las coordenadas de la ventana que vamos a grabar, win32con lo tengo por sus constantes

hwnd = findTopWindow("<Nombre Programa>")

esta linea busca una ventana que tenga de título y devuelve un handler

win32gui.SetWindowPos(hwnd, win32con.HWND_TOPMOST, 0,0,0,0,
win32con.SWP_NOMOVE | win32con.SWP_NOSIZE)

buscamos la posicion en x, y del handler y hace que esta posicion sea el nuevo (0,0) 

rect = win32gui.GetWindowPlacement(hwnd)[-1]

guardamos en una variable rect la pos (x,y)

while(True):

    img_np = np.array(img)
    cv2.imshow("frame", img_np)

    if key == 27:

guardamos en una variable img el rectangulo que habiamos identificado constantes guardo en una variable img_np el frame convertido en un array de numpy
